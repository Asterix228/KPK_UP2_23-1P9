# S8 — Subgroup Service (Сервис подгрупп)

Сервис управляет подгруппами внутри учебных групп (деление для иностранного языка, физкультуры и т.п.).
Хранит состав студентов в каждой подгруппе. **Не хранит** сами группы и профили студентов — они управляются соответствующими сервисами.

---

## Добавить подгруппу

Параметры, необходимые для создания подгруппы:

| Параметр    | Пояснение                         | Обязательность | Тип     | Ограничение                        | Значение по умолчанию |
|-------------|-----------------------------------|----------------|---------|------------------------------------|----------------------|
| name        | Название подгруппы                | Да             | string  | 1–100 символов                     | —                    |
| group_id    | ID учебной группы                 | Да             | integer | существующий id из Group Service   | —                    |
| purpose     | Назначение подгруппы              | Нет            | string  | до 200 символов                    | null                 |

**Уникальные комбинации параметров:**
- `(name, group_id)` — в одной группе не может быть двух подгрупп с одинаковым названием.

Данные, возвращаемые при успешном создании:

| Параметр    | Тип                |
|-------------|--------------------|
| id          | integer            |
| name        | string             |
| group_id    | integer            |
| purpose     | string или null    |
| is_active   | boolean            |
| created_at  | string (ISO 8601)  |
| updated_at  | string (ISO 8601)  |

---

## Изменить подгруппу по ID

Параметры для изменения (все необязательны):

| Параметр | Пояснение              | Обязательность | Тип     | Ограничение       |
|----------|------------------------|----------------|---------|-------------------|
| name     | Новое название         | Нет            | string  | 1–100 символов    |
| group_id | Новая учебная группа   | Нет            | integer | существующий id   |
| purpose  | Новое назначение       | Нет            | string  | до 200 символов   |

Возвращаемые данные — полный объект подгруппы (те же поля, что при создании).

| Параметр   | Тип               |
|------------|-------------------|
| id         | integer           |
| name       | string            |
| group_id   | integer           |
| purpose    | string или null   |
| is_active  | boolean           |
| created_at | string (ISO 8601) |
| updated_at | string (ISO 8601) |

---

## Удалить подгруппу по ID

Возвращает `true`, если подгруппа была деактивирована (`is_active = false`), иначе `false`. Физически запись из БД не удаляется.

---

## Получить подгруппу по ID

| Параметр    | Пояснение              | Тип               |
|-------------|------------------------|-------------------|
| id          | Идентификатор          | integer           |
| name        | Название               | string            |
| group_id    | ID учебной группы      | integer           |
| purpose     | Назначение             | string или null   |
| is_active   | Активна ли подгруппа   | boolean           |
| created_at  | Дата создания          | string (ISO 8601) |
| updated_at  | Дата последнего изм.   | string (ISO 8601) |

---

## Получить список подгрупп по параметрам

Параметры фильтрации:

| Параметр  | Пояснение                | Тип     |
|-----------|--------------------------|---------|
| group_id  | Фильтр по учебной группе | integer |
| is_active | Фильтр по активности     | boolean |
| limit     | Кол-во записей (1–100)   | integer |
| offset    | Смещение (≥ 0)           | integer |

Возвращается массив объектов подгрупп, каждый содержит:

| Параметр   | Тип               |
|------------|-------------------|
| id         | integer           |
| name       | string            |
| group_id   | integer           |
| purpose    | string или null   |
| is_active  | boolean           |
| created_at | string (ISO 8601) |
| updated_at | string (ISO 8601) |

---

## Добавить студента в подгруппу

| Параметр    | Пояснение          | Обязательность | Тип     | Ограничение             |
|-------------|-------------------|----------------|---------|-------------------------|
| subgroup_id | ID подгруппы       | Да             | integer | существующий id         |
| student_id  | ID студента        | Да             | integer | существующий id         |

Возвращаемые данные:

| Параметр    | Тип               |
|-------------|-------------------|
| id          | integer           |
| subgroup_id | integer           |
| student_id  | integer           |
| joined_at   | string (ISO 8601) |

---

## Удалить студента из подгруппы по ID записи

Возвращает `true`, если запись удалена, иначе `false`.

---

|-------------|---------------|---------|
| subgroup_id | ID подгруппы  | integer |

Возвращается массив объектов:

| Параметр   | Тип               |
|------------|-------------------|
| id         | integer           |
| student_id | integer           |
| joined_at  | string (ISO 8601) |

## ER-диаграмма

> Файл диаграммы: `erd.drawio` (открывается в [draw.io](https://app.diagrams.net/))

![ERD](erd.drawio)

```
Group          ──< Subgroup        (одна группа — много подгрупп)
Group          ──< Student         (одна группа — много студентов)
Subgroup  >──< Student  через SubgroupStudent  (многие ко многим)
```

| Таблица          | Назначение                                              |
|------------------|---------------------------------------------------------|
| Group            | Заглушка: учебная группа (управляется Group Service)    |
| Student          | Заглушка: студент (управляется Profile Service)         |
| Subgroup         | Подгруппа внутри учебной группы                         |
| SubgroupStudent  | Транзитивная таблица: состав студентов в подгруппах     |
